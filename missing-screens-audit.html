아이디/비밀번호 탭 + OTP 탭. KO/EN 언어 전환 포함.

🔑 비밀번호 찾기 누락
app/forgot-password.tsx — 없음

로그인 화면에 "비밀번호 찾기" 링크가 없고, 해당 화면 자체가 존재하지 않음. 현재는 OTP 탭으로 우회하지만 흐름이 불명확함.

우선순위: HIGH
📋 온보딩 / 스플래시 누락
app/onboarding.tsx — 없음

최초 설치 후 앱 소개나 권한 요청(푸시 알림, 위치) 안내 화면 없음. 로그인 직전에 필요.

우선순위: MEDIUM
홈 (Home)
🏠 홈 대시보드 구현 완료
app/(main)/index.tsx · MAI-01

날씨, 오늘 TBM 카드, 긴급조치 캐러셀(위험원/SOS/작업중지), 건의사항 미리보기.

⚠️ 위험원 보고 화면 누락
app/hazard/new.tsx — 없음

홈의 "위험원 보고" 카드를 누르면 onPress={() => {}} — 빈 핸들러. 실제 보고 입력 화면이 없음.

우선순위: HIGH
🚨 긴급 SOS 화면 누락
app/emergency/sos.tsx — 없음

SOS 카드 onPress도 빈 핸들러. 비상연락처 다이얼, 신고 확인 UI가 필요. 안전 앱에서 가장 중요한 기능 중 하나.

우선순위: HIGH
✋ 작업 중지 화면 누락
app/emergency/stop.tsx — 없음

"작업 중지" 카드 onPress도 빈 핸들러. 중지 사유 입력 및 관리자 알림 흐름이 없음.

우선순위: HIGH
TBM (Tool Box Meeting)
📋 TBM 목록 구현 완료
app/(main)/tbm.tsx · TBM-01

필터(전체/예정/완료/미완료), 오늘 히어로 카드, 과거 TBM 목록.

📄 오늘 TBM 세션 구현 완료
app/tbm/session/[id].tsx · TBM-02

날씨, 작업 정보, 위험 요인, 안전 조치 요약. 체크리스트로 이동.

✅ TBM 체크리스트 & 서명 구현 완료
app/tbm/checklist/[id].tsx

체크리스트 항목 확인, 서명 캡처, PDF 업로드.

🗂️ TBM 기록 상세 구현 완료
app/tbm/[id].tsx

완료된 과거 TBM 요약·서명 이력 열람.

➕ TBM 생성 화면 누락
app/tbm/new.tsx — 없음

관리자/현장반장이 새 TBM을 생성하는 화면이 없음. 목록에 생성 버튼이 있어야 하지만 현재 더미 데이터만 표시.

우선순위: HIGH
👥 참석자 관리 부분 구현
TBM 세션 내 참석자 필드만 존재

TBM 세션에 참석자 목록은 있으나, 참석자 추가/확인 전용 화면 없음. 작업자별 서명 확인 흐름도 불완전.

우선순위: MEDIUM

🔍 건의 상세 구현 완료
app/(main)/suggestions/[id].tsx

상세 내용 + 댓글 UI.

프로필 & 설정 (Profile / Settings)
⚙️ 설정 (알림) 부분 구현
app/(main)/profile.tsx

푸시 알림 ON/OFF 스위치만 있음. 프로필 정보(이름, 사번, 부서), 비밀번호 변경, 로그아웃 등 기본 설정 기능 없음.

우선순위: MEDIUM
🔔 알림 목록 누락
app/notifications.tsx — 없음

알림 ON/OFF는 있지만, 수신된 푸시 알림 이력을 볼 수 있는 인박스 화면이 없음.

우선순위: MEDIUM
우선순위별 구현 순서 제안
🔴 HIGH — 즉시 필요

1. ⚠️ 위험원 보고 화면 — 핵심 안전 기능, 빈 핸들러 상태
2. 🚨 긴급 SOS 화면 — 비상 연락 흐름 없음
3. ✋ 작업 중지 화면 — 핵심 안전 기능, 빈 핸들러
4. ➕ TBM 생성 화면 — 관리자가 TBM을 만들 방법 없음
5. 🔑 비밀번호 찾기 — 자격증명 복구 불가

🟡 MEDIUM — 단기 계획

6. 👥 참석자 관리 화면 — 서명 수집 흐름 완성
7. ⚙️ 프로필 편집 / 로그아웃 — 설정 화면 기능 확장
8. 🔔 알림 목록 화면 — 알림 이력 인박스
9. 📋 온보딩 — 최초 설치 안내
